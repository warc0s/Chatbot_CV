Proyecto: ChatBot sobre mi CV

Comparativa entre RAG vs Context Size

En este proyecto, he creado y evaluado dos tecnologías avanzadas de chatbots para determinar cuál ofrece un mejor resultado: RAG vs Context Size. Ambos chatbots están basados en mi información profesional, reflejando mi experiencia y conocimientos. Te invito a interactuar con ellos y comprobar cuál de ellos ofrece una mejor experiencia.
LLM + RAG [LlamaIndex]
Combina el LLM Llama 3.3 de 70B con capacidades de Recuperación de Información (RAG). En este caso, el RAG es mi CV en formato txt. Usa el framework LlamaIndex, y el embedding BGE-M3.
Entrar
Gemini Flash
Aprovecha la avanzada capacidad de contexto de Gemini Flash (1M de Tokens) para mantener conversaciones rápidas y precisas sobre el tema, en este caso, mi CV. Además, implementa un finetune de BERT para optimizar costes de LLM y bloquear preguntas no relacionadas.
